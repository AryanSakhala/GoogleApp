# GoogleApp
This app is build in R shiny

### App.csv file contains following column
* App
* Category
* Rating
* Reviews
* Size
* Installs
* Type
* Price
* Content Rating
* Genres
* Last Updated
* Current Version
* Android Version

## App has 3 main function:
* Plotting - Visualise
* Filtering - Filter
* Predicting - Prediction

### Visualisation is further categorised into 
* Price  Plot
* Category Plot
* Review Plot
